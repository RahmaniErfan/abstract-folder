# Changelog

## [Unreleased] - (Abstract Library Update)
### Added
- **Library Center**: New workspace leaf for decentralized knowledge package management.
- **Git Sync Engine**: Integrated `isomorphic-git` with mobile-compatible `ObsidianHttpAdapter`.
- **Abstract Bridge**: Virtual folder mounting system to merge remote libraries into the local vault view.
- **Contribution Engine**: Secure state machine for drafting and submitting changes to remote repositories.
- **GitHub Device Flow**: Passwordless OAuth integration for secure library access.

- **Marketplace & Registries**: Support for official and custom library registries.
- **Standalone Libraries**: Direct installation of Git repositories via URL.
- **Modular Settings**: Refactored settings UI into focused, maintainable sections under `src/ui/settings/`.
